# AI Resume Analyzer

A modern web application that analyzes resumes against job descriptions using AI technology. This tool helps job seekers optimize their resumes by identifying missing skills and providing actionable recommendations.

## Features

- **AI-Powered Analysis**: Uses OpenRouter API with GPT-3.5-turbo to intelligently compare resumes with job descriptions
- **Multiple File Formats**: Supports PDF and DOCX resume uploads
- **Comprehensive Feedback**: Provides strengths, missing skills, recommendations, and match percentage
- **Optimized Resume Generation**: Creates a professionally formatted resume tailored to the job description
- **Download Options**: Export optimized resume as PDF or DOCX files
- **Modern UI**: Beautiful, responsive interface built with Tailwind CSS
- **Real-time Analysis**: Get instant feedback on your resume
- **No API Key Required**: Uses pre-configured OpenRouter API key

## Tech Stack

- **Backend**: Python Flask
- **AI**: OpenRouter API (GPT-3.5-turbo)
- **Frontend**: HTML5, Tailwind CSS, JavaScript
- **File Processing**: PyPDF2, python-docx
- **PDF Generation**: ReportLab
- **Document Creation**: python-docx

## Prerequisites

- Python 3.7 or higher
- Internet connection for AI analysis

## Installation

1. **Clone or download the project**
   ```bash
   # If using git
   git clone <repository-url>
   cd Resume-Analyzer
   ```

2. **Create a virtual environment**
   ```bash
   python -m venv venv
   
   # On Windows
   venv\Scripts\activate
   
   # On Mac/Linux
   source venv/bin/activate
   ```

3. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

## Usage

1. **Start the application**
   ```bash
   python app.py
   ```

2. **Open your browser**
   Navigate to `http://localhost:5000`

3. **Analyze your resume**
   - Upload your resume (PDF or DOCX format)
   - Paste the job description
   - Click "Analyze Resume"
   - Review the analysis results
   - View the optimized resume generated by AI
   - Download the optimized resume as PDF or DOCX

## Configuration

The application uses OpenRouter API with a pre-configured API key. To use your own API key:

1. Open `app.py`
2. Find the line: `OPENROUTER_API_KEY = "YOUR_OPENROUTER_API_KEY_HERE"`
3. Replace with your actual OpenRouter API key

To get an OpenRouter API key:
1. Visit [OpenRouter.ai](https://openrouter.ai/)
2. Sign up or log in
3. Navigate to API Keys section
4. Create a new API key

## Analysis Features

The AI analyzer provides:

- **Match Percentage**: Overall compatibility score (0-100%)
- **Strengths**: Key skills and experiences that match the job
- **Missing Skills**: Important skills not found in your resume
- **Recommendations**: Specific suggestions to improve your resume
- **Key Matches**: Direct alignments between your resume and job requirements
- **Optimized Resume**: Professionally formatted resume tailored to the job description
- **Download Options**: Export optimized resume in PDF or DOCX format

## File Support

- **PDF**: `.pdf` files up to 16MB
- **Word Documents**: `.docx` and `.doc` files up to 16MB

## Security & Privacy

- Resume files are temporarily processed and immediately deleted after analysis
- API keys are used locally and never stored on the server
- No personal data is retained after the analysis session

## Troubleshooting

### Common Issues

1. **"Could not extract text from file"**
   - Ensure your PDF is text-based (not scanned images)
   - Try saving the Word document as a different format

2. **"AI analysis failed"**
   - Check your OpenAI API key is valid
   - Ensure you have sufficient API credits
   - Verify your internet connection

3. **File upload errors**
   - Check file size is under 16MB
   - Ensure file format is supported (PDF, DOCX, DOC)

### Getting Help

If you encounter issues:

1. Check the console for error messages
2. Verify all dependencies are installed correctly
3. Ensure your OpenAI API key has sufficient credits
